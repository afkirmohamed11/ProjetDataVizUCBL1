# Projet DataViz UCBL1

## Branche Académique

Bienvenue dans la branche académique du projet **DataViz UCBL1**. Cette branche est dédiée aux travaux académiques réalisés dans le cadre du projet. Vous y trouverez des rapports, des documents de planification, et des analyses liés à la réalisation du projet.

### Contenu
- **Rapports académiques** : Documentation détaillée sur les étapes de réalisation du projet.
- **Planification** : Informations sur la gestion et l'organisation du projet.
- **Analyses** : Résultats et études effectués dans le cadre du projet.

Cette branche reflète l'aspect méthodologique et académique du projet, mettant en avant les efforts de recherche et de développement réalisés par l'équipe.
